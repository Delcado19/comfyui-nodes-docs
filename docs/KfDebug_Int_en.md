# Documentation
- Class name: KfDebug_Int
- Category: Debug
- Output node: True
- Repo Ref: https://github.com/dmarx/ComfyUI-Keyframed

The KfDebug_Int node is designed to facilitate users in inspecting and analyzing integer values within a computation graph. It enables monitoring and understanding the flow of integer data, ensuring correctness of operations involving integers in the model. The node aids debugging by providing a clear, focused view of integer handling and transformation, avoiding complexity from other data types.

# Input types
## Required
- input
    - The input parameter is a key element for feeding integer values into the KfDebug_Int node. It is critical to the node's functionality, as it allows the node to monitor and analyze integer values. The input plays a decisive role in determining the node's output and the insights it provides during debugging.
    - Comfy dtype: INT
    - Python dtype: int

# Output types
- output
    - The output of the KfDebug_Int node is the inspected and analyzed processed integer value. It serves as confirmation of the node's operation and proof of the integrity of integer values in the computation graph. The output is important, as it provides a foundation for further debugging or validating model performance.
    - Comfy dtype: INT
    - Python dtype: int
