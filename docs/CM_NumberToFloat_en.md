# Documentation
- Class name: NumberToFloat
- Category: math/conversion
- Output node: False
- Repo Ref: https://github.com/evanspearman/ComfyMath

The NumberToFloat node is designed to convert an input number into floating-point format. It plays a critical role in ensuring that numerical computations across different data types maintain precision and consistency.

# Input types
## Required
- a
    - The 'a' parameter is the input number that the node intends to convert to a floating-point value. It is essential to the node's operation, as it directly affects the numerical representation of the output.
    - Comfy dtype: NUMBER
    - Python dtype: Union[int, float, str]

# Output types
- result
    - The 'result' output is the converted floating-point number derived from the input. It is important because it represents the primary output of the node and the final result of the conversion process.
    - Comfy dtype: FLOAT
    - Python dtype: float

# Usage tips
- Infra type: CPU
