The BatchCLIPSeg node uses the CLIPSeg model to segment single or batch images. It guides the segmentation process through natural language descriptions, combining images and text descriptions to generate segmentation masks. This node can also optionally apply thresholding and mask manipulation techniques.

# Input types
## Required
- images
    - The image to be segmented. This is the core input of the segmentation process, providing visual data.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor
- text
    - The text description used to guide the segmentation process. This description aligns the segmentation results with user intent.
    - Comfy dtype: STRING
    - Python dtype: str
- threshold
    - The threshold used to determine segmentation sensitivity. Adjusting this threshold affects the fineness of the segmentation.
    - Comfy dtype: FLOAT
    - Python dtype: float
- binary_mask
    - Determines whether the output mask is binarized. If true, the mask is simplified to a binary form.
    - Comfy dtype: BOOLEAN
    - Python dtype: bool
- combine_mask
    - Controls whether masks of multiple objects are merged into a single mask. This affects the output, potentially merging multiple segmentation regions.
    - Comfy dtype: BOOLEAN
    - Python dtype: bool
- use_cuda
    - Indicates whether to use CUDA for processing. On compatible hardware, this can significantly speed up the segmentation process.
    - Comfy dtype: BOOLEAN
    - Python dtype: bool

# Output types
- Mask
    - The segmentation masks generated by the node. These masks correspond to regions of interest defined by the input image and text description.
    - Comfy dtype: MASK
    - Python dtype: torch.Tensor

    def segment_image(self, images, text, threshold, binary_mask, combine_mask, use_cuda):        
        from transformers import CLIPSegProcessor, CLIPSegForImageSegmentation
        out = []
        height, width, _ = images[0].shape
        if use_cuda and torch.cuda.is_available():
            device = torch.device("cuda")
        else:
            device = torch.device("cpu")
        dtype = comfy.model_management.unet_dtype()
        model = CLIPSegForImageSegmentation.from_pretrained("CIDAS/clipseg-rd64-refined")
        model.to(dtype)
        model.to(device)
        images = images.to(device)
        processor = CLIPSegProcessor.from_pretrained("CIDAS/clipseg-rd64-refined")
        pbar = comfy.utils.ProgressBar(images.shape[0])
        autocast_condition = (dtype != torch.float32) and not comfy.model_management.is_device_mps(device)
        with torch.autocast(comfy.model_management.get_autocast_device(device), dtype=dtype) if autocast_condition else nullcontext():
            for image in images:
                image = (image* 255).type(torch.uint8)
                prompt = text
                input_prc = processor(text=prompt, images=image, return_tensors="pt")
                # Move the processed input to the device
                for key in input_prc:
                    input_prc[key] = input_prc[key].to(device)
                
                outputs = model(**input_prc)
    
                tensor = torch.sigmoid(outputs[0])
                tensor_thresholded = torch.where(tensor > threshold, tensor, torch.tensor(0, dtype=torch.float))
                tensor_normalized = (tensor_thresholded - tensor_thresholded.min()) / (tensor_thresholded.max() - tensor_thresholded.min())
                tensor = tensor_normalized

                out.append(resized_tensor)
          
        results = torch.stack(out).cpu().float()
        
        if combine_mask:
            combined_results = torch.max(results, dim=0)[0]
            results = combined_results.unsqueeze(0).repeat(len(images),1,1)

            results = results.round()
