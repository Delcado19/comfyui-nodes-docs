# Documentation
- Class name: SDXLPromptStylerbyEnvironment
- Category: Style Prompts
- Output node: False
- Repo Ref: https://github.com/wolfden/ComfyUi_PromptStylers

SDXLPromptStylerbyEnvironment class aims to stylize prompts based on predefined sets of styles and templates. It accepts positive and negative text inputs as well as a selected style, and generates stylized prompts accordingly. This node also provides an option to log prompts for review.

# Input types
## Required
- text_positive
    - text_positive parameter is a string representing the positive content of the prompt. It is essential for the node to generate stylized positive prompts and plays an important role in the overall output.
    - Comfy dtype: STRING
    - Python dtype: str
- text_negative
    - text_negative parameter is a string representing the negative content of the prompt. It is used to create stylized negative prompts and is essential for the node's functionality.
    - Comfy dtype: STRING
    - Python dtype: str
- style
    - style parameter determines the stylization method applied to the prompt. It is a string selected from a list of predefined styles and is essential for the node's operation.
    - Comfy dtype: STRING
    - Python dtype: str
- log_prompt
    - log_prompt parameter is an optional option that allows users to decide whether to log the generated prompt. It affects the node's behavior by enabling or disabling the logging feature.
    - Comfy dtype: COMBO['No', 'Yes']
    - Python dtype: str

# Output types
- positive_prompt_text_g
    - positive_prompt_text_g output is the stylized positive prompt text generated by the node. It is an important result of the node execution and reflects the positive content of the input.
    - Comfy dtype: STRING
    - Python dtype: str
- negative_prompt_text_g
    - negative_prompt_text_g output is the stylized negative prompt text generated by the node. It represents the node's output for the negative content of the input prompt.
    - Comfy dtype: STRING
    - Python dtype: str
