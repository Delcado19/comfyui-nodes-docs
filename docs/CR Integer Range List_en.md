# Documentation
- Class name: CR_IntegerRangeList
- Category: Comfyroll/List
- Output node: False
- Repo Ref: https://github.com/RockOfFire/ComfyUI_Comfyroll_CustomNodes

The CR_IntegerRangeList node generates a list of integers within a specified range, with customizable start, end, and step parameters. It can also adapt to a "ping-pong" behavior, reversing the range in alternating iterations for more varied output. This node plays a key role when a sequence of integer values is needed for further processing or iteration.

# Input types
## Required
- start
- The 'start' parameter defines the beginning of the integer range. It is crucial because it determines the starting point of the sequence generated by the node, affecting the overall output sequence.
    - Comfy dtype: INT
    - Python dtype: int
- end
- The 'end' parameter specifies the end of the integer range. It is critical because it determines the last number in the generated sequence, thereby affecting the range's length.
    - Comfy dtype: INT
    - Python dtype: int
- step
- The 'step' parameter sets the increment between each integer in the range. It is important because it controls the spacing between numbers, affecting the density of the generated sequence.
    - Comfy dtype: INT
    - Python dtype: int
- loops
- The 'loops' parameter determines how many times the range is repeated or iterated. It is important because it affects the total number of numbers produced by the node.
    - Comfy dtype: INT
    - Python dtype: int
- ping_pong
- The 'ping_pong' parameter introduces an alternating mode where the range is reversed each iteration. It is important because it can introduce variability into the sequence, which may be necessary for some applications.
    - Comfy dtype: BOOLEAN
    - Python dtype: bool

# Output types
- INT
- The 'INT' output provides a list of integers generated within the specified range based on the input parameters. It is important because it represents the node's primary output, ready for further use or analysis.
