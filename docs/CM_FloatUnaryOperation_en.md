# Documentation
- Class name: FloatUnaryOperation
- Category: math/float
- Output node: False
- Repo Ref: https://github.com/evanspearman/ComfyMath

The FloatUnaryOperation node applies a mathematical operation to a single floating-point number, converting it to another floating-point value. It is designed to handle various unary operations, making it flexible for mathematical calculations in data processing pipelines.

# Input types
## Required
- op
    - The parameter 'op' specifies the unary mathematical operation to perform. It is critical because it determines the type of transformation applied to the input floating-point value.
    - Comfy dtype: str
    - Python dtype: str
- a
    - The parameter 'a' represents the input floating-point value on which the unary operation will be applied. It is a fundamental component of the node's operation, as it is the subject of the mathematical transformation.
    - Comfy dtype: float
    - Python dtype: float

# Output types
- result
    - The output 'result' provides the transformed floating-point value after applying the specified unary operation. It is important because it represents the result of the mathematical calculation performed by the node.
    - Comfy dtype: float
    - Python dtype: float
