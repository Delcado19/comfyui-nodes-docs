# Documentation
- Class name: StringTrim
- Display name: Trim Text
- Category: text
- Output node: False
- Repo Ref: https://github.com/comfyanonymous/ComfyUI

Removes leading and trailing whitespace from a string based on the selected mode.

# Input types
## Required
- string
    - The input text to be trimmed.
    - Comfy dtype: STRING
- mode
    - Selects which whitespace characters to remove from the string.
    - Comfy dtype: COMBO

# Output types
- STRING
    - The trimmed text with specified whitespace characters removed.
    - Comfy dtype: STRING
