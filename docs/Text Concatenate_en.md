The WAS_Text_Concatenate node is designed to efficiently merge multiple text inputs into a single string. It achieves this by joining the provided text inputs using a specified separator, optionally trimming leading and trailing whitespace for cleaner output. This node plays a key role in simplifying text processing workflows by streamlining the aggregation of text data.

# Input types
## Required
- delimiter
    - The separator parameter defines the character or string used when joining individual text inputs. It is critical for determining the format of the final output string and influences the readability and structure of the combined text.
    - Comfy dtype: STRING
    - Python dtype: str
## Optional
- clean_whitespace
    - The clean_whitespace parameter is an optional flag that, when set to 'true', instructs the node to strip any leading and trailing whitespace from text inputs before concatenation. This ensures the resulting text has no unnecessary spaces and is neatly formatted.
    - Comfy dtype: COMBO['true', 'false']

    - The text_a parameter represents one of the optional text inputs that can be provided to the node. It contributes to the final merged text as part of the concatenation process. Including this parameter allows multiple distinct text segments to be added to the combined output.

    - Python dtype: str
- text_b
    - The text_b parameter serves a similar purpose to text_a, allowing another distinct text segment to be included during concatenation. It enhances the node's ability to handle various text inputs and create a comprehensive final text.

class WAS_Text_Concatenate:

    def __init__(self):
        pass

    @classmethod
    def INPUT_TYPES(cls):
        return {'required': {'delimiter': ('STRING', {'default': ', '}), 'clean_whitespace': (['true', 'false'],)}, 'optional': {'text_a': (TEXT_TYPE, {'forceInput': True if TEXT_TYPE == 'STRING' else False}), 'text_b': (TEXT_TYPE, {'forceInput': True if TEXT_TYPE == 'STRING' else False}), 'text_c': (TEXT_TYPE, {'forceInput': True if TEXT_TYPE == 'STRING' else False}), 'text_d': (TEXT_TYPE, {'forceInput': True if TEXT_TYPE == 'STRING' else False})}}
    RETURN_TYPES = (TEXT_TYPE,)
    FUNCTION = 'text_concatenate'
    CATEGORY = 'WAS Suite/Text'

    def text_concatenate(self, delimiter, clean_whitespace, **kwargs):
        text_inputs: list[str] = []
        if delimiter == '\\n':
            delimiter = '\n'
        for k in sorted(kwargs.keys()):
            v = kwargs[k]
            if isinstance(v, str):
                if clean_whitespace == 'true':
                    v = v.strip()
                if v != '':
                    text_inputs.append(v)
        merged_text = delimiter.join(text_inputs)
        return (merged_text,)
