# Documentation
- Class name: CreateFluidMask
- Category: KJNodes/masking/generate
- Output node: False
- Repo Ref: https://github.com/kijai/ComfyUI-KJNodes.git

CreateFluidMask node is designed to generate fluid animation effects, which can be used to create dynamic and visually appealing mask overlays. It utilizes fluid dynamics principles to simulate inflow and dye motion within a specified resolution and duration, generating a series of images and their corresponding masks.

# Input types
## Required
- frames
    - 'frames' parameter determines the total number of frames to generate for the fluid mask animation. It is crucial because it determines the length of the animation sequence and influences the overall visual narrative through fluid motion.
    - Comfy dtype: INT
    - Python dtype: int
- width
    - 'width' parameter sets the width of the generated fluid mask in pixels. It is an important parameter because it defines the spatial resolution along the x-axis, affecting the detail and quality of the output.
    - Comfy dtype: INT
    - Python dtype: int
- height
    - 'height' parameter specifies the height of the generated fluid mask in pixels. It plays an important role in establishing the vertical resolution of the output, affecting the level of detail visible in the animation.
    - Comfy dtype: INT
    - Python dtype: int
## Optional
- invert
    - 'invert' parameter allows inversion of the generated mask. When set to True, it flips the mask, creating a contrasting effect for visual results.
    - Comfy dtype: BOOLEAN
    - Python dtype: bool
- inflow_count
    - 'inflow_count' parameter determines the number of inflow points to consider in the fluid simulation. It affects the complexity and distribution of fluid flow, contributing to the overall mask aesthetics.
    - Comfy dtype: INT
    - Python dtype: int
- inflow_velocity
    - 'inflow_velocity' parameter sets the velocity of the inflow points, affecting how quickly the fluid moves and disperses in the simulation. It is a key factor in controlling the dynamics of the fluid animation.
    - Comfy dtype: INT
    - Python dtype: int
- inflow_radius
    - 'inflow_radius' parameter defines the influence radius of each inflow point, determining the area affected by the inflow. It plays a crucial role in shaping the appearance of the fluid mask.
    - Comfy dtype: INT
    - Python dtype: int
- inflow_padding
    - 'inflow_padding' parameter adds padding around the inflow points to prevent fluid from directly hitting the edges of the mask. It helps create smoother transitions for the fluid flow at the boundaries.
    - Comfy dtype: INT
    - Python dtype: int
- inflow_duration
    - 'inflow_duration' parameter specifies the duration for which the inflow will be active during the simulation. It is essential for controlling the timing of the fluid effect within the animation sequence.
    - Comfy dtype: INT
    - Python dtype: int

# Output types
- IMAGE
    - 'IMAGE' output provides the sequence of images generated by the fluid simulation. Each image represents a frame in the animation, showing the fluid movement and dye dispersion over time.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor
- MASK
    - 'MASK' output consists of binary masks corresponding to each frame in the 'IMAGE' output. These masks depict the areas where fluid is present, providing a clear distinction between the fluid and the background.
    - Comfy dtype: MASK
    - Python dtype: torch.Tensor
