# Input types

    - Comfy dtype: INT
    - Python dtype: int

- height
    - Height of the image.
    - Comfy dtype: INT
    - Python dtype: int

- original_size
    - Original dimensions of the image.
    - Comfy dtype: BOX
    - Python dtype: list
