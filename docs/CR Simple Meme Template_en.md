# Documentation
- Class name: CR_SimpleMemeTemplate
- Category: Comfyroll/Graphics/Template
- Output node: False
- Repo Ref: https://github.com/RockOfFire/ComfyUI_Comfyroll_CustomNodes

CR_SimpleMemeTemplate is a node for generating custom meme templates. It allows users to input an image, select a preset for the meme structure, and add text to the top and bottom of the image. The node also provides customization options for font, text color, and adding decorative bars above and below the text. The primary purpose of this node is to facilitate the creation of meme images using a simple and user-friendly interface, making it accessible to a broad user base for meme creation.

# Input types
## Required
- image
    - The image parameter is required because it defines the base image on which the meme template will be applied. The choice of image directly affects the final appearance of the meme, making it a critical element in the node's operation.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor
- preset
    - The preset parameter determines the structure and predefined text of the meme template. It simplifies the customization process by allowing users to choose from common meme formats.
    - Comfy dtype: COMBO['custom', 'One Does Not Simply ... MEME IN COMFY', 'This is fine.', 'Good Morning ... No Such Thing!']
    - Python dtype: str
- text_top
    - The text_top parameter allows users to input the text that will appear at the top of the meme image. It plays an important role in conveying the intended message or theme of the meme.
    - Comfy dtype: STRING
    - Python dtype: str
- text_bottom
    - The text_bottom parameter is used to display text at the bottom of the meme image. It complements the top text and contributes to the overall composition of the meme.
    - Comfy dtype: STRING
    - Python dtype: str
- font_name
    - The font_name parameter enables users to select the font style for the text on the meme. The choice of font can significantly affect the readability and visual appeal of the meme, making it an important customization option.
    - Comfy dtype: COMBO[file_list]
    - Python dtype: str
- max_font_size
    - The max_font_size parameter sets the maximum size for the text font in the meme template. It is a critical factor in determining the readability and visual balance of the text within the meme layout.
    - Comfy dtype: INT
    - Python dtype: int
- font_color
    - The font_color parameter specifies the color of the text in the meme template. It is a key aspect of the node's functionality, as it affects the visibility of the text and the overall visual impact of the meme.
    - Comfy dtype: COMBO[COLORS]
    - Python dtype: str
- font_outline
    - The font_outline parameter adds an outline effect to the text, which can improve the visibility of the text against the meme image background. The choice of outline style can also add a decorative element to the meme's design.
    - Comfy dtype: COMBO['none', 'thin', 'thick', 'extra thick']
    - Python dtype: str
- bar_color
    - The bar_color parameter determines the color of the decorative bars that may be added above and below the text in the meme template. This helps draw attention to the text and provides a contrasting background.
    - Comfy dtype: COMBO[COLORS]
    - Python dtype: str
- bar_options
    - The bar_options parameter determines whether decorative bars are added to the meme template and their position relative to the text. This can affect the overall design of the meme and the emphasis on the text.
    - Comfy dtype: COMBO['no bars', 'top', 'bottom', 'top and bottom']
    - Python dtype: str

# Output types
- image
    - The image output contains the final meme template generated by the node. It represents the culmination of all input parameters and customizations made by the user.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor
- show_help
    - The show_help output provides a link to the documentation or help page associated with the node. This may be useful for users seeking additional guidance or information on how to use the node.
    - Comfy dtype: STRING
    - Python dtype: str
