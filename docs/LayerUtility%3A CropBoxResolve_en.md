# Documentation
- Class name: LayerUtility: CropBoxResolve
- Category: 😺dzNodes/LayerUtility
- Output node: False
- Repo Ref: https://github.com/chflame163/ComfyUI_LayerStyle

Decomposes a crop box into its individual coordinate and dimension values. Extracts the x, y, width, and height from a single BOX input.

# Input types
## Required
- crop_box
    - The input crop box to resolve into individual values.
    - Comfy dtype: BOX

# Output types
- x
    - The horizontal position of the crop box.
    - Comfy dtype: INT
- y
    - The vertical position of the crop box.
    - Comfy dtype: INT
- width
    - The width of the crop box.
    - Comfy dtype: INT
- height
    - The height of the crop box.
    - Comfy dtype: INT
