# Documentation
- Class name: PolyexponentialScheduler
- Category: sampling/custom_sampling/schedulers
- Output node: False
- Repo Ref: https://github.com/comfyanonymous/ComfyUI

The PolyexponentialScheduler node is designed to generate a custom schedule for sigma values in diffusion models. It uses a polyexponential function to smoothly vary the noise level over a given number of steps. This scheduler is crucial for controlling learning dynamics and ensuring stable sampling results.

# Input types
## Required
- steps
    - The "steps" parameter defines the number of intervals over which sigma values will be scheduled. It is crucial as it directly affects the granularity and duration of the noise schedule, influencing the overall sampling process.
    - Comfy dtype: INT
    - Python dtype: int
- sigma_max
    - The "sigma_max" parameter represents the maximum value of sigma, i.e., the initial noise level in the diffusion process. It is important as it sets the upper bound of the noise level, thereby affecting the convergence and quality of the final sample.
    - Comfy dtype: FLOAT
    - Python dtype: float
- sigma_min
    - The "sigma_min" parameter represents the minimum value of sigma, corresponding to the target noise level at the end of the diffusion process. It is crucial as it determines the final noise level, affecting the clarity and accuracy of the generated sample.
    - Comfy dtype: FLOAT
    - Python dtype: float
## Optional
- rho
    - The "rho" parameter adjusts the curvature of the polyexponential schedule. It influences the rate at which sigma values decrease from "sigma_max" to "sigma_min", thereby affecting the overall shape of the noise schedule and sampling dynamics.
    - Comfy dtype: FLOAT
    - Python dtype: float

# Output types
- SIGMAS
    - The output 'SIGMAS' is a sequence of sigma values generated by the PolyexponentialScheduler. These values are critical to the sampling process as they dictate the noise level at each step, directly impacting the quality and stability of the resulting samples.
    - Comfy dtype: FLOAT
    - Python dtype: torch.Tensor
