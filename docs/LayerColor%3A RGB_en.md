Adjust the RGB channels of the image.

# Input types
## Required

- image
    - Input image.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor

- R
    - Adjustment value for the red channel.
    - Comfy dtype: INT
    - Python dtype: int

- G 
    - Adjustment value for the green channel.
    - Comfy dtype: INT
    - Python dtype: int
   
- B
    - Adjustment value for the blue channel.
    - Comfy dtype: INT
    - Python dtype: int
    

# Output types

- image
    - Output image.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor
