# Documentation
- Class name: StringReplace
- Display name: Replace Text
- Category: text
- Output node: False
- Repo Ref: https://github.com/comfyanonymous/ComfyUI

Performs a find-and-replace operation on the input string. Replaces each occurrence of the 'find' substring with the 'replace' substring.

# Input types
## Required
- string
    - The input text to perform the replacement on.
    - Comfy dtype: STRING
- find
    - The substring to search for within the input string.
    - Comfy dtype: STRING
- replace
    - The substring to replace each occurrence of the 'find' substring with.
    - Comfy dtype: STRING

# Output types
- STRING
    - The resulting string after performing the replacement.
    - Comfy dtype: STRING
