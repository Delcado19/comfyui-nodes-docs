
# Documentation
- Class name: ttN pipeLoader_v2
- Category: ttN/pipe
- Output node: False
- Repo Ref: https://github.com/ttN/ComfyUI-ttN-nodes

This node is used to load and initialize pipelines for various tasks within the tinyterraNodes framework, facilitating the setup and configuration of data processing and model interaction pipelines.

# Input types
## Required
- ckpt_name

## Optional
- model_override
    - Allows overriding the default model, supporting the use of alternative models in the pipeline.
    - Comfy dtype: MODEL
    - Python dtype: str
- clip_override
    - Allows overriding the default CLIP model, integrating alternative visual understanding capabilities.
    - Comfy dtype: CLIP
    - Python dtype: str
- optional_lora_stack
    - Optional parameter for specifying LoRA adjustment stacks, enhancing model performance through custom configurations.
    - Comfy dtype: LORA_STACK

    - Comfy dtype: CONTROLNET_STACK
