Select a color on the color swatch and output it. Modified from the web extensions of mtb nodes, thanks to the original author.

# Input types

## Required

- color
    - Input color.
    - Comfy dtype: COLOR
    - Python dtype: str
    - Options: {"default": "#FFFFFF"}

- mode
    - Output mode
    - Comfy dtype: STRING
    - Python dtype: str
    - Options: ['HEX', 'DEC']

# Output types

- value
    - Output color.
    - Comfy dtype: STRING
    - Python dtype: str
