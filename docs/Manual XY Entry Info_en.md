
# Documentation
- Class name: Manual XY Entry Info
- Category: Efficiency Nodes/XY Inputs
- Output node: False

The Manual XY Entry Info node is designed to provide detailed information for manual XY chart entry in specific scenarios, facilitating the understanding and manipulation of XY data to achieve customized plotting and analysis. By receiving user-input annotation information, this node enhances the customization and specificity of XY chart data, providing richer context for subsequent plotting processes.

# Input types
## Required
- notes
    - The notes input allows users to provide additional context or information related to manual XY entry, thereby enhancing the degree of customization and specificity in the plotting process. This feature enables users to add personalized descriptions to XY data, improving data interpretability and usability.
