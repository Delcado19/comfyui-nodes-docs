
# Documentation
- Class name: XYAny
- Category: List Stuff
- Output node: False

The XYAny node aims to generate combinations of X and Y values, and optionally extend these combinations by adding a third dimension Z. It helps create and label these combinations, providing fallback labels and the ability to specify custom labels. This node is particularly suitable for data manipulation and preparation tasks that require exploring or visualizing relationships or interactions between different sets of data points.

# Input types
## Required
- X
    - Main set of values for combination. Acts as one axis for generating value pairs.
    - Comfy dtype: *
    - Python dtype: List[Any]
- Y
    - Secondary set of values for combination with X values. Acts as the other axis for generating value pairs.
    - Comfy dtype: *
    - Python dtype: List[Any]
- X_Label_Fallback
    - Fallback label for X values, used when no specific label is provided. Ensures each X value has an associated label.
    - Comfy dtype: COMBO[STRING]
    - Python dtype: List[str]
- Y_Label_Fallback
    - Fallback label for Y values, used when no specific label is provided. Ensures each Y value has an associated label.
    - Comfy dtype: COMBO[STRING]
    - Python dtype: List[str]
- Z_Label_Fallback
    - Fallback label for Z values, used when no specific label is provided. Allows labeling the third dimension of data.
    - Comfy dtype: COMBO[STRING]
    - Python dtype: List[str]
## Optional
- Z
    - Optional third set of values for combination with X and Y values, adding a third dimension to the combination.
    - Comfy dtype: *
    - Python dtype: Optional[List[Any]]
- X_Labels
    - Custom label for X values, if provided, overrides the fallback label.
    - Comfy dtype: *
    - Python dtype: Optional[List[Any]]
- Y_Labels
    - Custom label for Y values, if provided, overrides the fallback label.
    - Comfy dtype: *
    - Python dtype: Optional[List[Any]]
- Z_Labels
    - Custom label for Z values, if provided, overrides the fallback label.
    - Comfy dtype: *
    - Python dtype: Optional[List[Any]]
