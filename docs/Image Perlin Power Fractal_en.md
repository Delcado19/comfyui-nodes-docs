# Documentation
- Class name: WAS_Image_Perlin_Power_Fractal
- Category: WAS Suite/Image/Generate/Noise
- Output node: False
- Repo Ref: https://github.com/WASasquatch/was-node-suite-comfyui

The WAS_Image_Perlin_Power_Fractal node programmatically generates fractal noise patterns, which can be used to create complex textures. It uses the Perlin noise algorithm to produce natural, harmonious, seamless noise structures that can tile infinitely. This node is especially suitable for generating textures for 3D models, game environments, or any application that requires coherent and repeatable textures.

# Input types
## Required
- width
- The Width parameter defines the texture width in pixels. It is crucial for setting the horizontal resolution of the output image and affects the overall size of the texture.
    - Comfy dtype: INT
    - Python dtype: int
- height
- The Height parameter determines the vertical extent of the generated texture in pixels. Together with the Width parameter, it defines the full resolution of the output image.
    - Comfy dtype: INT
    - Python dtype: int
- scale
- The Scale parameter influences the level of detail in the generated texture. Higher scale values produce more detailed, finer noise patterns, while lower values yield coarser, broader patterns.
    - Comfy dtype: INT
    - Python dtype: int
- octaves
- The Octaves parameter controls the number of frequencies used to generate the fractal noise. More octaves create more complex and detailed textures but increase computational cost.
    - Comfy dtype: INT
    - Python dtype: int
- persistence
- The Persistence parameter defines how much each successive octave contributes to the overall noise pattern. It affects the contrast between different noise layers.
    - Comfy dtype: FLOAT
    - Python dtype: float
- lacunarity
- The Lacunarity parameter sets the frequency gap between successive octaves. Higher lacunarity values produce more fragmented and detailed textures at smaller scales.
    - Comfy dtype: FLOAT
    - Python dtype: float
- exponent
- The Exponent parameter adjusts the amplitude of each octave in the fractal noise. It shapes the overall noise pattern and can be used to create finer textures.
    - Comfy dtype: FLOAT
    - Python dtype: float
## Optional
- seed
- The Seed parameter is used to generate deterministic noise patterns. It ensures that the same seed always produces the same texture, promoting consistency across multiple images or applications.
    - Comfy dtype: INT
    - Python dtype: int

# Output types
- image
- The Output Image is a seamless, tileable texture generated by the Perlin power fractal function. It can be used directly in graphics applications or further processed as needed.
    - Comfy dtype: IMAGE
    - Python dtype: PIL.Image.Image
