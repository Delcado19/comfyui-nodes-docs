# Documentation
- Class name: PulidFluxModelLoader
- Display name: Load PuLID Flux Model
- Category: pulid
- Output node: False
- Repo Ref: https://github.com/sipie800/ComfyUI-PuLID-Flux-Enhanced

Loads a PuLID Flux model from a specified file, preparing it for use in further processing.

# Input types
## Required
- pulid_file
    - Selects the PuLID Flux model file to load.
    - Comfy dtype: COMBO
    - Options: populated at runtime from installed files

# Output types
- PULIDFLUX
    - The loaded PuLID Flux model object.
    - Comfy dtype: PULIDFLUX
