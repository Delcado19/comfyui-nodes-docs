# Documentation
- Class name: UnetLoaderGGUF
- Display name: Unet Loader (GGUF)
- Category: bootleg
- Output node: False
- Repo Ref: https://github.com/city96/ComfyUI-GGUF

Loads a UNet model from a GGUF file, outputting the model for use in further nodes.

# Input types
## Required
- unet_name
    - Selects the specific GGUF model file to load from available options.
    - Comfy dtype: COMBO
    - Options: populated at runtime from installed files

# Output types
- MODEL
    - The loaded UNet model, ready for use in other nodes.
    - Comfy dtype: MODEL
