# Documentation
- Class name: KfDebug_Float
- Category: Debugging
- Output node: True
- Repo Ref: https://github.com/dmarx/ComfyUI-Keyframed

The KfDebug_Float node is designed to facilitate the debugging process by providing a method to inspect and analyze floating-point numbers in data processing pipelines. It is a critical tool for developers to verify the integrity and behavior of numerical data at various stages of computation.

# Input types
## Required
- input_float
    - The input_float parameter is essential for the node's operation, as it represents the floating-point number that needs to be debugged. It plays a central role in the node's execution, serving as the primary data to be inspected and analyzed.
    - Comfy dtype: FLOAT
    - Python dtype: float

# Output types
- output_float
    - The output_float parameter is important because it represents the processed and debugged floating-point number from the node. It is crucial for downstream processes, as it ensures the continuity and reliability of the data flow.
    - Comfy dtype: FLOAT
    - Python dtype: float

# Usage tips
- Infra type: CPU
