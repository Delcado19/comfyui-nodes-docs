# Documentation
- Class name: SDXLPromptbyContemporaryNordicArt
- Category: Style Prompts
- Output node: False
- Repo Ref: https://github.com/wolfden/ComfyUi_PromptStylers

The SDXLPromptbyContemporaryNordicArt node is designed to generate and stylize prompts based on contemporary Nordic art style. It creatively combines positive and negative text inputs with selected art styles to produce aesthetically heuristic prompts usable for various creative or descriptive purposes.

# Input types
## Required
- text_positive
    - The text_positive parameter is the node's key input, providing a positive background or affirmation that will be artistically integrated into the prompt. It significantly influences the tone and content of the generated prompt, ensuring alignment with the desired positive message.
    - Comfy dtype: STRING
    - Python dtype: str
- text_negative
    - The text_negative parameter allows inclusion of a negative background, enabling artistic contrast or transformation within the prompt. Though optional, it adds depth to the final output by offering a perspective opposite to the positive text.
    - Comfy dtype: STRING
    - Python dtype: str
- style
    - The style parameter is required because it determines the specific contemporary Nordic art style applied to the prompt. The style choice directly affects the aesthetic and thematic elements of the generated prompt, guiding its overall artistic direction.
    - Comfy dtype: STRING
    - Python dtype: str
- log_prompt
    - The log_prompt parameter is an optional toggle; when set to 'Yes', it enables logging of the prompt generation process. This is useful for debugging or reviewing the intermediate steps taken to create the final prompt.
    - Comfy dtype: COMBO['No', 'Yes']
    - Python dtype: str

# Output types
- positive_prompt_text_g
    - The positive_prompt_text_g output contains the stylized positive prompt text generated by the node. It represents the artistic fusion of the input text with the selected Nordic art style and can be used in various applications.
    - Comfy dtype: STRING
    - Python dtype: str
- negative_prompt_text_g
    - The negative_prompt_text_g output provides the stylized negative prompt text, complementing the positive prompt by offering a contrasting or transformative perspective. It is a component of the node's output that contributes to the completeness of the generated prompts.
    - Comfy dtype: STRING
    - Python dtype: str
