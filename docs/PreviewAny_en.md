# Documentation
- Class name: PreviewAny
- Display name: Preview as Text
- Category: utils
- Output node: True
- Repo Ref: https://github.com/comfyanonymous/ComfyUI

Displays the input as a text preview in the UI while passing it through as a STRING output. Useful for inspecting intermediate values during workflow debugging.

# Input types
## Required
- source
    - Any input value to be displayed as text and forwarded as STRING.
    - Comfy dtype: *

# Output types
- STRING
    - The input value converted to a STRING, passed through for downstream use.
    - Comfy dtype: STRING
