# Documentation
- Class name: LoadImageOutput
- Display name: Load Image (from Outputs)
- Category: image
- Output node: False
- Repo Ref: https://github.com/comfyanonymous/ComfyUI

Load an image from the output folder. When the refresh button is clicked, the node will update the image list and automatically select the first image, allowing for easy iteration.

# Input types
## Required
- image
    - Comfy dtype: COMBO

# Output types
- IMAGE
    - Comfy dtype: IMAGE
- MASK
    - Comfy dtype: MASK
