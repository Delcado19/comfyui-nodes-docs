# Documentation
- Class name: PrimitiveStringMultiline
- Display name: Text String (Multiline)
- Category: utils/primitive
- Output node: False
- Repo Ref: https://github.com/comfyanonymous/ComfyUI

A utility node that holds and passes through a multiline text string. It acts as a primitive string constant or passthrough in a ComfyUI workflow.

# Input types
## Required
- value
    - The multiline text string to store or forward.
    - Comfy dtype: STRING

# Output types
- STRING
    - The same multiline text string passed through unchanged.
    - Comfy dtype: STRING
