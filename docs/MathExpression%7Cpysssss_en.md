# Documentation
- Class name: MathExpression|pysssss
- Display name: Math Expression 🐍
- Category: utils
- Output node: True
- Repo Ref: https://github.com/pythongosssss/ComfyUI-Custom-Scripts

Evaluates a mathematical expression using optional variables a, b, c. Outputs both integer and float representations of the result. Displays the result in the UI as a side effect.

# Input types
## Required
- expression
    - The mathematical expression to evaluate, referencing variables a, b, c (e.g., 'a + b * c').
    - Comfy dtype: STRING
## Optional
- a
    - Optional value assigned to variable 'a' for use in the expression.
    - Comfy dtype: *
- b
    - Optional value assigned to variable 'b' for use in the expression.
    - Comfy dtype: *
- c
    - Optional value assigned to variable 'c' for use in the expression.
    - Comfy dtype: *

# Output types
- INT
    - The integer result of the evaluated expression.
    - Comfy dtype: INT
- FLOAT
    - The floating-point result of the evaluated expression.
    - Comfy dtype: FLOAT
