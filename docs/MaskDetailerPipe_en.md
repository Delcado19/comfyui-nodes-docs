# Documentation
- Class name: MaskDetailerPipe
- Category: ImpactPack/Detailer
- Output node: False
- Repo Ref: https://github.com/ltdrdata/ComfyUI-Impact-Pack.git

The MaskDetailerPipe node processes and enhances image details based on a provided mask. It uses advanced models and algorithms to refine and enhance masked regions, focusing on improving visual quality and detail of the specified area.

# Input types
## Required
- image
    - Input image, processed by the node. It serves as the basis for all enhancement and refinement operations.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor
- mask
    - Mask defining the region of interest within the image. It guides the enhancement process to focus on specific parts of the image.
    - Comfy dtype: MASK
    - Python dtype: torch.Tensor
- basic_pipe
    - A set of pretrained models and configurations for initial image processing and analysis.
    - Comfy dtype: BASIC_PIPE
    - Python dtype: Tuple[torch.nn.Module, torch.nn.Module, Any, Any, Any]
## Optional
- guide_size
    - Guidance size parameter that affects the scale of detail enhancement. It determines the level of detail the node will focus on during processing.
    - Comfy dtype: FLOAT
    - Python dtype: float

# Output types
- enhanced_img_batch
    - A batch of enhanced images generated by the processing. It includes refined details as specified by the input mask and parameters.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor
- cropped_enhanced_list
    - A series of cropped and enhanced image segments. Each segment corresponds to a region defined by the input mask and has been processed to enhance detail.
    - Comfy dtype: COMBO[IMAGE]
    - Python dtype: List[torch.Tensor]

# Usage tips
- Infra type: GPU
