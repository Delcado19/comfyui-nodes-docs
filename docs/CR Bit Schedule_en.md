# Documentation
- Class name: CR_BitSchedule
- Category: Comfyroll/Animation/Schedule
- Output node: False
- Repo Ref: https://github.com/RockOfFire/ComfyUI_Comfyroll_CustomNodes

The CR_BitSchedule node is designed to generate a schedule based on a binary string input, capable of specifying intervals and loop counts. It processes the binary string to create a sequence of scheduled events, which is particularly useful for animation and timing arrangements in projects.

# Input types
## Required
- binary_string
    - The binary string is a sequence of binary digits (0s and 1s) that the node uses to create the schedule. Each bit in the string corresponds to an event in the schedule, and the schedule will repeat according to the specified number of loops.
    - Comfy dtype: STRING
    - Python dtype: str
- interval
    - The interval parameter determines the spacing between scheduled events in the sequence. It is an integer value that directly affects the timing of the schedule.
    - Comfy dtype: INT
    - Python dtype: int
## Optional
- loops
    - The loop parameter specifies how many times the schedule should repeat. It is an optional integer that allows customization of the schedule's duration.
    - Comfy dtype: INT
    - Python dtype: int

# Output types
- SCHEDULE
    - The output schedule is a string representing the sequence of scheduled events generated based on the input binary string, interval, and loop count. It is used for further processing or display within the application.
    - Comfy dtype: STRING
    - Python dtype: str
- show_text
    - The show_text output provides a URL pointing to help documentation that offers additional information and guidance on how to use the schedule generated by the node.
    - Comfy dtype: STRING
    - Python dtype: str

class CR_BitSchedule:

    @classmethod
    def INPUT_TYPES(s):
        return {'required': {'binary_string': ('STRING', {'multiline': True, 'default': ''}), 'interval': ('INT', {'default': 1, 'min': 1, 'max': 99999}), 'loops': ('INT', {'default': 1, 'min': 1, 'max': 99999})}}

    FUNCTION = 'bit_schedule'
    CATEGORY = icons.get('Comfyroll/Animation/Schedule')

    def bit_schedule(self, binary_string, interval, loops=1):
        show_help = 'https://github.com/Suzie1/ComfyUI_Comfyroll_CustomNodes/wiki/Schedule-Nodes#cr-bit-schedule'
        schedule = []
        binary_string = binary_string.replace(' ', '').replace('\n', '')
        '\n        for i in range(len(binary_string) * loops):\n            index = i % len(binary_string)  # Use modulo to ensure the index continues in a single sequence\n            bit = int(binary_string[index])\n            schedule.append(f"{i},{bit}")\n        '
        for i in range(len(binary_string) * loops):
            schedule_index = i * interval
            bit_index = i % len(binary_string)
            bit = int(binary_string[bit_index])
            schedule.append(f'{schedule_index},{bit}')
        schedule_out = '\n'.join(schedule)
        return (schedule_out, show_help)
