# Documentation
- Class name: FloatToNumber
- Category: math/conversion
- Output node: False
- Repo Ref: https://github.com/evanspearman/ComfyMath

The FloatToNumber node is designed to convert floating-point values into a more general numeric representation. It plays a key role in data type conversion, ensuring compatibility between different parts of mathematical or computational workflows.

# Input types
## Required
- a
    - The 'a' parameter is a floating-point number that the node is designed to convert. It is essential for the operation of the node as it determines the input data type that needs to be converted.
    - Comfy dtype: FLOAT
    - Python dtype: float

# Output types
- result
    - The 'result' output is the number converted from the input floating-point value. It signifies the successful conversion from a specific data type to a more general numeric form.
    - Comfy dtype: NUMBER
    - Python dtype: number
