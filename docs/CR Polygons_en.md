# Documentation
- Class name: CR_Polygons
- Category: Comfyroll/Graphics/Pattern
- Output node: False
- Repo Ref: https://github.com/RockOfFire/ComfyUI_Comfyroll_CustomNodes

CR_Polygons is a node for generating geometric patterns composed of hexagons or triangles. It allows customization of polygon colors, background, and line attributes to create visually distinct patterns. This node is especially suitable for creating background textures or complex designs for various applications.

# Input types
## Required
- mode
- The mode parameter determines the shape of the polygons to draw, which can be hexagons or triangles. This choice significantly influences the overall pattern generated by the node, giving users control over the geometric style of the output.
    - Comfy dtype: COMBO['hexagons', 'triangles']
    - Python dtype: str
- width
- The width parameter sets the pattern's width in pixels. It is a key factor in determining the pattern's proportion and directly affects the final size of the output image.
    - Comfy dtype: INT
    - Python dtype: int
- height
- The height parameter sets the pattern's height in pixels. It works with the width to determine the overall size of the pattern, which is essential for adapting the design to specific display requirements.
    - Comfy dtype: INT
    - Python dtype: int
- rows
- The rows parameter specifies the number of polygon rows to draw within the pattern. It affects polygon density and layout, influencing the complexity of the final design.
    - Comfy dtype: INT
    - Python dtype: int
- columns
- The columns parameter determines the number of polygon columns to draw within the pattern. It works with the rows parameter to define the overall grid structure of the pattern.
    - Comfy dtype: INT
    - Python dtype: int
- face_color
- The fill color parameter defines the color used to fill the interior of the polygons. It plays an important role in determining the visual appeal and thematic consistency of the pattern.
    - Comfy dtype: COLORS
    - Python dtype: str
- background_color
- The background color parameter sets the color behind the polygons. It is crucial for providing contrast and ensuring the pattern stands out against its background.
    - Comfy dtype: COLORS
    - Python dtype: str
- line_color
- The line color parameter determines the color of the lines outlining the polygons. It is a key aspect for defining pattern boundaries and enhancing overall visual clarity.
    - Comfy dtype: COLORS
    - Python dtype: str
- line_width
- The line width parameter specifies the thickness of the lines used to draw polygon outlines. It influences the prominence of pattern edges and contributes to the aesthetic of the overall design.
    - Comfy dtype: INT
    - Python dtype: int
## Optional
- face_color_hex
- The fill color hex parameter allows custom hexadecimal color codes for the interior of the polygons. It provides users flexibility to fine‑tune the pattern's color scheme.
    - Comfy dtype: STRING
    - Python dtype: str
- bg_color_hex
- The background color hex parameter allows specifying a custom hexadecimal color code for the background. It ensures users can customize the background color to match their design preferences.
    - Comfy dtype: STRING
    - Python dtype: str
- line_color_hex
- The line color hex parameter allows using custom hexadecimal color codes to define the polygon lines. It gives users the ability to adjust line colors for the desired visual effect.
    - Comfy dtype: STRING
    - Python dtype: str

class CR_Polygons:

    @classmethod
    def INPUT_TYPES(s):
        modes = ['hexagons', 'triangles']
        return {'required': {'mode': (modes,), 'width': ('INT', {'default': 512, 'min': 64, 'max': 4096}), 'height': ('INT', {'default': 512, 'min': 64, 'max': 4096}), 'rows': ('INT', {'default': 5, 'min': 1, 'max': 512}), 'columns': ('INT', {'default': 5, 'min': 1, 'max': 512}), 'face_color': (COLORS,), 'background_color': (COLORS,), 'line_color': (COLORS,), 'line_width': ('INT', {'default': 2, 'min': 0, 'max': 512})}, 'optional': {'face_color_hex': ('STRING', {'multiline': False, 'default': '#000000'}), 'bg_color_hex': ('STRING', {'multiline': False, 'default': '#000000'}), 'line_color_hex': ('STRING', {'multiline': False, 'default': '#000000'})}}
    RETURN_TYPES = ('IMAGE', 'STRING')
    RETURN_NAMES = ('IMAGE', 'show_help')
    FUNCTION = 'draw'
    CATEGORY = icons.get('Comfyroll/Graphics/Pattern')

    def draw(self, mode, width, height, rows, columns, face_color, background_color, line_color, line_width, face_color_hex='#000000', bg_color_hex='#000000', line_color_hex='#000000'):
        if face_color == 'custom':
            face_color = face_color_hex
        if line_color == 'custom':
            line_color = line_color_hex
        if background_color == 'custom':
            background_color = bg_color_hex
        (fig, ax) = plt.subplots(figsize=(width / 100, height / 100))
        fig.set_facecolor(background_color)
        plt.xlim(0, width / 100)
        plt.ylim(0, height / 100)
        plt.axis('off')
        plt.tight_layout(pad=0, w_pad=0, h_pad=0)
        plt.autoscale(False)
        if mode == 'hexagons':
            vertices = 6
        elif mode == 'triangles':
            vertices = 3
        cell_width = width / 100 / columns
        cell_height = width / height * np.sqrt(3) * (height / 100) / (2 * columns)
        for row in range(rows + 2):
            for col in range(columns + 2):
                x = col * cell_width
                y = row * cell_height
                if row % 2 == 1:
                    x += cell_width / 2
                hexagon = RegularPolygon((x, y), numVertices=vertices, radius=cell_width / 1.732, edgecolor=line_color, linewidth=line_width, facecolor=face_color)
                ax.add_patch(hexagon)
        img_buf = io.BytesIO()
        plt.savefig(img_buf, format='png')
        img = Image.open(img_buf)
        image_out = pil2tensor(img.convert('RGB'))
        show_help = 'https://github.com/Suzie1/ComfyUI_Comfyroll_CustomNodes/wiki/Pattern-Nodes#cr-polygons'
        return (image_out, show_help)
