# Documentation
- Class name: XYtoPercent
- Category: 😺dzNodes/LayerUtility/Data
- Output node: False
- Repo Ref: https://github.com/chflame163/ComfyUI_LayerStyle

Convert absolute coordinates to percentage coordinates.


# Input types

## Required


- background_image
    - Background image.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor

- layer_image
    - Layer image.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor

- x
    - x coordinate.
    - Comfy dtype: INT
    - Python dtype: int

- y
    - y coordinate.
    - Comfy dtype: INT
    - Python dtype: int

# Output types

- x_percent
    - x percentage coordinate.
    - Comfy dtype: FLOAT
    - Python dtype: float   

- x_percent
    - y percentage coordinate.
    - Comfy dtype: FLOAT
    - Python dtype: float
