# Documentation
- Class name: WAS_Integer_Place_Counter
- Category: WAS Suite/Integer
- Output node: False
- Repo Ref: https://github.com/WASasquatch/was-node-suite-comfyui

This node is designed to calculate the number of digits in an integer value. It serves as a foundational tool for applications that require digit counting operations, such as numerical analysis or data processing.

# Input types
## Required
- int_input
    - Specify the integer input whose digit count is to be determined. It is the primary data element of the node operation and directly influences the result.
    - Comfy dtype: INT
    - Python dtype: int

# Output types
- INT_PLACES
    - Output represents the digit count of the provided integer. It is important for applications that need to know the length of a number's digits.
    - Comfy dtype: INT
    - Python dtype: int

# Usage tips
- Infra type: CPU
