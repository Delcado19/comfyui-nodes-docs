# Documentation
- Class name: MikeySampler
- Category: Mikey/Sampling
- Output node: False
- Repo Ref: https://github.com/bash-j/mikey_nodes

MikeySampler node aims to generate high-quality samples from the given base model and refinement model, while utilizing VAE for latent space manipulation. It performs complex sampling techniques and combines conditional inputs to optimize the output. This node also upscales the generated images according to the specified method and strength, ensuring that the final output meets the desired resolution and quality standards.

# Input types
## Required
- seed
    - The seed parameter is crucial for the reproducibility of the sampling process, ensuring that samples generated in different runs are consistent. It plays an important role in controlling the randomness of the sampling algorithm.
    - Comfy dtype: INT
    - Python dtype: int
- base_model
    - The base model is the fundamental neural network used for initial sampling in latent space. It is a key component that determines the quality and diversity of generated samples.
    - Comfy dtype: MODEL
    - Python dtype: torch.nn.Module
- refiner_model
    - The refinement model is used to further improve the quality of samples generated by the base model. It refines the latent space representation to achieve higher fidelity in the final output.
    - Comfy dtype: MODEL
    - Python dtype: torch.nn.Module
- samples
    - The samples parameter represents the number of latent vectors to sample from the base model. It directly affects the diversity of the generated output.
    - Comfy dtype: LATENT
    - Python dtype: torch.Tensor
## Optional
- upscale_by
    - The upscale_by parameter determines the scale factor for the generated image. It is an important factor in controlling the final output resolution.
    - Comfy dtype: FLOAT
    - Python dtype: float
- hires_strength
    - The hires_strength parameter adjusts the strength of the high-resolution upscaling process. It affects the sharpness and detail level of the upscaled image.
    - Comfy dtype: FLOAT
    - Python dtype: float

# Output types
- latent
    - The latent parameter represents the encoded latent vector obtained during the sampling process. It is important because it serves as an intermediate representation between the model and the final upscaled image.
    - Comfy dtype: LATENT
    - Python dtype: torch.Tensor

class MikeySampler:

    @classmethod
    def INPUT_TYPES(s):
        return {'required': {'base_model': ('MODEL',), 'refiner_model': ('MODEL',), 'samples': ('LATENT',), 'vae': ('VAE',), 'positive_cond_base': ('CONDITIONING',), 'negative_cond_base': ('CONDITIONING',), 'positive_cond_refiner': ('CONDITIONING',), 'negative_cond_refiner': ('CONDITIONING',), 'model_name': (folder_paths.get_filename_list('upscale_models'),), 'seed': ('INT', {'default': 0, 'min': 0, 'max': 18446744073709551615}), 'upscale_by': ('FLOAT', {'default': 1.0, 'min': 0.0, 'max': 10.0, 'step': 0.1}), 'hires_strength': ('FLOAT', {'default': 1.0, 'min': 0.0, 'max': 2.0, 'step': 0.1})}}
    RETURN_TYPES = ('LATENT',)
    FUNCTION = 'run'
    CATEGORY = 'Mikey/Sampling'

    def adjust_start_step(self, image_complexity, hires_strength=1.0):
        image_complexity /= 24
        if image_complexity > 1:
            image_complexity = 1
        image_complexity = min([0.55, image_complexity]) * hires_strength
        return min([16, 16 - int(round(image_complexity * 16, 0))])

    def run(self, seed, base_model, refiner_model, vae, samples, positive_cond_base, negative_cond_base, positive_cond_refiner, negative_cond_refiner, model_name, upscale_by=1.0, hires_strength=1.0, upscale_method='normal'):
        image_scaler = ImageScale()
        vaeencoder = VAEEncode()
        vaedecoder = VAEDecode()
        uml = UpscaleModelLoader()
        upscale_model = uml.load_model(model_name)[0]
        iuwm = ImageUpscaleWithModel()
        sample1 = common_ksampler(base_model, seed, 25, 6.5, 'dpmpp_2s_ancestral', 'simple', positive_cond_base, negative_cond_base, samples, start_step=0, last_step=18, force_full_denoise=False)[0]
        sample2 = common_ksampler(refiner_model, seed, 30, 3.5, 'dpmpp_2m', 'simple', positive_cond_refiner, negative_cond_refiner, sample1, disable_noise=True, start_step=21, force_full_denoise=True)
        if upscale_by == 0:
            return sample2
        else:
            sample2 = sample2[0]
        pixels = vaedecoder.decode(vae, sample2)[0]
        (org_width, org_height) = (pixels.shape[2], pixels.shape[1])
        img = iuwm.upscale(upscale_model, image=pixels)[0]
        (upscaled_width, upscaled_height) = (int(org_width * upscale_by // 8 * 8), int(org_height * upscale_by // 8 * 8))
        img = image_scaler.upscale(img, 'nearest-exact', upscaled_width, upscaled_height, 'center')[0]
        if hires_strength == 0:
            return (vaeencoder.encode(vae, img)[0],)
        image_complexity = calculate_image_complexity(img)
        start_step = self.adjust_start_step(image_complexity, hires_strength)
        latent = vaeencoder.encode(vae, img)[0]
        out = common_ksampler(base_model, seed, 16, 9.5, 'dpmpp_2m_sde', 'karras', positive_cond_base, negative_cond_base, latent, start_step=start_step, force_full_denoise=True)
        return out
