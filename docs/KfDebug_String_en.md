# Documentation
- Class name: KfDebug_String
- Category: Debugging
- Output node: True
- Repo Ref: https://github.com/dmarx/ComfyUI-Keyframed

This node is designed to facilitate the inspection and analysis of string data within the workflow. It enables users to view and verify the content and structure of strings at different stages, ensuring the accuracy and completeness of the text information being processed.

# Input types
## Required
- input_string
    - The input_string parameter is crucial for this node because it carries the text data to be inspected. It is a key component since the node's main function revolves around analyzing and displaying the content of this string.
    - Comfy dtype: STRING
    - Python dtype: str

# Output types
- output_string
    - output_string represents the result of the node's operation, usually identical to input_string but may be modified or annotated based on the debugging process. It is important because it provides the final inspected text data.
    - Comfy dtype: STRING
    - Python dtype: str
