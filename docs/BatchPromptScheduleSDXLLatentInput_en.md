# Documentation
- Class name: BatchPromptScheduleEncodeSDXLLatentInput
- Category: FizzNodes 📅🅕🅝/BatchScheduleNodes
- Output node: False
- Repo Ref: https://github.com/FizzleDorf/ComfyUI_FizzNodes

BatchPromptScheduleEncodeSDXLLatentInput node is designed to process and encode a batch of prompts into latent representations suitable for use in generative models. It handles the complexity of prompt interpolation and conditioning, abstracting away the details of the process and providing a simplified interface for generating latent inputs.

# Input types

- width

    - Comfy dtype: INT
    - Python dtype: int
- height
    - The height parameter, similar to width, is essential for specifying the vertical dimensions of the input image, affecting how the node processes and encodes image data.
    - Comfy dtype: INT
    - Python dtype: int
- text_g
    - The text_g input is a string containing the prompt for the generative model. It is a key component of the node's functionality as it directly influences the generation of latent representations.
    - Comfy dtype: STRING
    - Python dtype: str
- num_latents
    - The num_latents parameter specifies the number of latent vectors generated by the node. It is important because it determines the output size of the latent space representation.
    - Comfy dtype: LATENT
    - Python dtype: Dict[str, torch.Tensor]

# Output types
- POS
    - The POS output provides positive conditioning data for the generative model, which is important for steering the generation process toward desired outcomes.
    - Comfy dtype: CONDITIONING
    - Python dtype: torch.Tensor
- NEG
    - The NEG output contains negative conditioning data, used to guide the generation process away from undesired results.
    - Comfy dtype: CONDITIONING
    - Python dtype: torch.Tensor
- LATENT
    - The LATENT output represents the latent vectors generated by the node, used as input for further processing or directly in the generative model.
    - Comfy dtype: LATENT
    - Python dtype: Dict[str, torch.Tensor]
