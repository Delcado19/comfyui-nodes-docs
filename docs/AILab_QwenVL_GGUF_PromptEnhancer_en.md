# Documentation
- Class name: AILab_QwenVL_GGUF_PromptEnhancer
- Display name: QwenVL Prompt Enhancer (GGUF)
- Category: 🧪AILab/QwenVL
- Output node: False
- Repo Ref: https://github.com/1038lab/ComfyUI-QwenVL

Enhances a text prompt using a QwenVL GGUF model, optionally forcing the output to English. Supports preset or custom system prompts and standard generation parameters.

# Input types
## Required
- model_name
    - GGUF model entry defined in gguf_models.json.
    - Comfy dtype: COMBO
    - Default: `"Qwen3-4B-Q4_K_M.gguf"`
    - Options: populated at runtime from installed files
- prompt_text
    - Prompt text to enhance. Leave blank to just emit the preset instruction.
    - Comfy dtype: STRING
    - Default: `""`
- preset_system_prompt
    - Predefined system prompt to guide the enhancement behavior.
    - Comfy dtype: COMBO
    - Default: `"📝 Enhance"`
    - Options: `📝 Enhance`, `📝 Refine`, `📝 Creative Rewrite`, `📝 Detailed Visual`, `📝 Artistic Style`, `📝 Technical Specs`
- custom_system_prompt
    - Custom system prompt string; overrides the preset if provided.
    - Comfy dtype: STRING
    - Default: `""`
- max_tokens
    - Maximum number of tokens to generate in the output.
    - Comfy dtype: INT
    - Default: `256`
- temperature
    - Controls randomness of the output; lower values make results more deterministic.
    - Comfy dtype: FLOAT
    - Default: `0.7`
- top_p
    - Nucleus sampling parameter; restricts sampling to top cumulative probability mass.
    - Comfy dtype: FLOAT
    - Default: `0.9`
- repetition_penalty
    - Penalizes repeated tokens to reduce redundancy in output.
    - Comfy dtype: FLOAT
    - Default: `1.1`
- english_output
    - Force final output in English using translation prompt.
    - Comfy dtype: BOOLEAN
    - Default: `false`
- device
    - Select device; auto prefers GPU when available.
    - Comfy dtype: COMBO
    - Default: `"auto"`
    - Options: `auto`, `cuda`, `cpu`, `mps`
- seed
    - Random seed for reproducible generation.
    - Comfy dtype: INT
    - Default: `1`

# Output types
- ENHANCED_OUTPUT
    - The enhanced text output generated by the QwenVL model.
    - Comfy dtype: STRING
