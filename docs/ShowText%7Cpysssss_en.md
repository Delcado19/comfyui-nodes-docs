# Documentation
- Class name: ShowText|pysssss
- Display name: Show Text 🐍
- Category: utils
- Output node: True
- Repo Ref: https://github.com/pythongosssss/ComfyUI-Custom-Scripts

Displays the input text in the UI for debugging or inspection purposes. It passes the text through unchanged as an output string, allowing it to be used downstream while also making it visible to the user.

# Input types
## Required
- text
    - The string to be displayed and passed through.
    - Comfy dtype: STRING

# Output types
- STRING
    - The same text string, passed through unchanged for use by downstream nodes.
    - Comfy dtype: STRING
    - Returns a list
