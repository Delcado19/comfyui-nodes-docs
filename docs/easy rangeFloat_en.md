# Documentation
- Class name: RangeFloat
- Category: EasyUse/Logic/Type
- Output node: False
- Repo Ref: https://github.com/yolain/ComfyUI-Easy-Use.git

This node generates a series of floating‑point numbers within a specified range, based on user‑defined mode and parameters. It abstracts the range creation process, allowing precise control over start value, end value, step size, as well as step count and endpoint inclusion.

# Input types
## Required
- range_mode
- The mode parameter determines the logic used to generate the range sequence. It is required because it sets the operational context for the node, deciding whether the range is generated by a fixed ‘step size’ or a specified ‘step count’.
    - Comfy dtype: COMBO[String]
    - Python dtype: str
- start
- The start parameter defines the initial value of the range. It is critical because it establishes the baseline for the generated sequence, affecting the entire output.
    - Comfy dtype: FLOAT
    - Python dtype: decimal.Decimal
- stop
- The end parameter marks the terminal point of the range. It is important because, together with the start value, it defines the boundaries of the generated sequence.
    - Comfy dtype: FLOAT
    - Python dtype: decimal.Decimal
- step
- The step size parameter determines the increment between consecutive values in the range sequence. Its value directly influences the granularity and pattern of the generated series.
    - Comfy dtype: FLOAT
    - Python dtype: decimal.Decimal
- end_mode
- The end mode parameter decides whether the terminal value is included in the range sequence. This affects endpoint inclusion, thereby influencing the composition of the output.
    - Comfy dtype: COMBO[String]
    - Python dtype: str
## Optional
- num_steps
- The step count parameter specifies the desired number of steps in the range sequence. It is important because it directly determines how many values the node produces.
    - Comfy dtype: INT
    - Python dtype: int

# Output types
- range
- The range output provides the generated floating‑point number series, which is the core result of the node's operation. It encapsulates the essence of the range generation process.
    - Comfy dtype: LIST[FLOAT]
    - Python dtype: List[float]
- range_sizes
- The range size output provides the count of values in each individually generated range sequence, offering insight into the distribution and structure of the output.
    - Comfy dtype: LIST(INT)
    - Python dtype: List[int]
