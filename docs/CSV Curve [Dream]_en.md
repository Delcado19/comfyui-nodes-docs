The CSV Curve node is designed to parse and apply animation curves based on CSV file data. It dynamically generates animation parameters by parsing CSV file content, supporting interpolation and custom CSV parsing processes. This node flexibly converts external data into animation curves, providing creators with a powerful and precise method to control animation changes and flow.

# Input types
## Required
- frame_counter
    - This parameter tracks the current frame of the animation, which is crucial for synchronizing the animation curve with the overall animation sequence. It ensures that the curve application stays consistent with the animation progress.
    - Comfy dtype: FRAME_COUNTER
    - Python dtype: FrameCounter
- csvfile
    - Specifies the CSV file path used to generate the animation curve. This allows the node to dynamically interpret and apply the data contained in the file, providing flexible control sources for animation.
    - Comfy dtype: STRING
    - Python dtype: str
- first_column_type
    - Determines whether the first column of the CSV file represents seconds or frames, affecting how the data is interpreted in the animation time context. Correctly setting this parameter is critical for accurately mapping data to the animation timeline.
    - Comfy dtype: COMBO[STRING]
    - Python dtype: List[str]
- interpolate
    - Controls whether interpolation is performed between data points in the CSV file. Enabling interpolation generates smoother animation curves, improving the fluidity and naturalness of the animation.
    - Comfy dtype: COMBO[STRING]
    - Python dtype: List[str]
- csv_dialect
    - Specifies the dialect of the CSV file, enabling the node to accurately parse files with different format conventions. This enhances compatibility, allowing the node to handle various CSV formats.
    - Comfy dtype: COMBO[STRING]
    - Python dtype: List[str]

# Output types
- FLOAT
    - Floating-point value generated by the CSV Curve node, representing a point on the animation curve. This provides high-precision control over animation parameters.
    - Comfy dtype: FLOAT
    - Python dtype: float
- INT
    - Integer representation of the generated animation curve point, offering an alternative precision level for animation parameters requiring discrete values.
    - Comfy dtype: INT

class DreamCSVCurve:
    NODE_NAME = "CSV Curve"

    @classmethod
    def INPUT_TYPES(cls):
        return {
            "required": SharedTypes.frame_counter | {
                "csvfile": ("STRING", {"default": "", "multiline": False}),
                "first_column_type": (["seconds", "frames"],),
                "interpolate": (["true", "false"],),
                "csv_dialect": (csv.list_dialects(),)
            },
        }

    CATEGORY = NodeCategories.ANIMATION_CURVES
    RETURN_TYPES = ("FLOAT", "INT")
    RETURN_NAMES = ("FLOAT", "INT")
    FUNCTION = "result"

    @classmethod
    def IS_CHANGED(cls, *values):
        return hashed_as_strings(*values)

    def _row_yield(self, file, csv_dialect):
        prev_row = None
        for row in csv.reader(file, dialect=csv_dialect):
            if len(row) == 2 and _is_as_float(row[0]) and _is_as_float(row[1]):
                row = list(map(float, row))
                yield (prev_row, row)
                prev_row = row
        if prev_row is not None:
            yield (prev_row, None)

    def result(self, csvfile, frame_counter: FrameCounter, first_column_type, interpolate, csv_dialect):
        interpolate = interpolate == "true"

        def _first_col_to_frame(v: float):
            if first_column_type == "frames":
                return round(v)
            else:
                return round(v * frame_counter.frames_per_second)

        with open(csvfile) as f:
            for (prev, current) in self._row_yield(f, csv_dialect):
                if prev is None and frame_counter.current_frame < _first_col_to_frame(current[0]):
                    # before first row
                    return (current[1], int(round(current[1])))
                if current is None:
                    # after last row
                    return (prev[1], int(round(prev[1])))
                if prev is not None and current is not None:
                    frame1 = _first_col_to_frame(prev[0])
                    value1 = prev[1]
                    frame2 = _first_col_to_frame(current[0])
                    value2 = current[1]
                    if frame1 <= frame_counter.current_frame and interpolate and frame2 > frame_counter.current_frame:
                        offset = (frame_counter.current_frame - frame1) / float(frame2 - frame1)
                        v = value1 * (1.0 - offset) + value2 * offset
                        return (v, int(round(v)))
                    elif frame1 <= frame_counter.current_frame and frame2 > frame_counter.current_frame:
                        return (value1, int(round(value1)))
        return (0.0, 0)
