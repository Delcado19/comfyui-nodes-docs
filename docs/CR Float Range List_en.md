# Documentation
- Class name: CR_FloatRangeList
- Category: Comfyroll/List
- Output node: False
- Repo Ref: https://github.com/RockOfFire/ComfyUI_Comfyroll_CustomNodes

The CR_FloatRangeList node is designed to generate a list of floating-point numbers within a specified range, with optional math operations and formatting applied. It provides customization options through parameters such as start value, end value, step size, and operation, offering users a versatile tool for creating numeric sequences for specific needs.

# Input types
## Required
- start
    - The start value parameter defines the initial value of the range. It is very important because it sets the starting point for the number sequence generated by the node, significantly affecting the overall output.
    - Comfy dtype: FLOAT
    - Python dtype: float
- end
    - The end value parameter specifies the final value of the range. It is critical because it determines the end point of the sequence, directly affecting the length and the last value of the generated list.
    - Comfy dtype: FLOAT
    - Python dtype: float
- step
    - The step size parameter determines the increment between each value in the range. It plays an important role in the density of the numeric sequence, controlling the number of generated values and their distribution.
    - Comfy dtype: FLOAT
    - Python dtype: float
- operation
    - The operation parameter allows applying trigonometric functions to each value in the range. This can transform the sequence, introduce non-linear characteristics, and enable more complex patterns.
    - Comfy dtype: COMBO[none, sin, cos, tan]
    - Python dtype: str
- decimal_places
    - The decimal places parameter controls the precision of floating-point numbers by specifying the number of digits after the decimal point. It is important for controlling the granularity of the output values.
    - Comfy dtype: INT
    - Python dtype: int
- ignore_first_value
    - The ignore first value parameter determines whether to exclude the initial value of the range from the output. This can be important when the starting point is not of interest or needs to be omitted in subsequent processing.
    - Comfy dtype: BOOLEAN
    - Python dtype: bool
- max_values_per_loop
    - The max value per loop parameter sets the maximum number of values generated per loop iteration. It is a key factor in controlling the total number of values produced by the node.
    - Comfy dtype: INT
    - Python dtype: int
- loops
    - The loop count parameter specifies the number of times the range operation should be repeated. It is important for creating multiple sequences or extending the output list.
    - Comfy dtype: INT
    - Python dtype: int
- ping_pong
    - The ping-pong parameter introduces a behavior where every other loop sequence is reversed, producing a 'ping-pong' effect. This can be used to generate symmetrical or oscillating patterns.
    - Comfy dtype: BOOLEAN
    - Python dtype: bool

class CR_FloatRangeList:

    @classmethod
    def INPUT_TYPES(s):
        operations = ['none', 'sin', 'cos', 'tan']
        return {'required': {'start': ('FLOAT', {'default': 0.0, 'min': -99999.99, 'max': 99999.99, 'step': 0.01}), 'end': ('FLOAT', {'default': 1.0, 'min': -99999.99, 'max': 99999.99, 'step': 0.01}), 'step': ('FLOAT', {'default': 1.0, 'min': -99999.99, 'max': 99999.99, 'step': 0.01}), 'operation': (operations,), 'decimal_places': ('INT', {'default': 2, 'min': 0, 'max': 10}), 'ignore_first_value': ('BOOLEAN', {'default': True}), 'max_values_per_loop': ('INT', {'default': 128, 'min': 1, 'max': 99999}), 'loops': ('INT', {'default': 1, 'min': 1, 'max': 999}), 'ping_pong': ('BOOLEAN', {'default': False})}}
    RETURN_TYPES = ('FLOAT', 'STRING')
    RETURN_NAMES = ('FLOAT', 'show_help')
    OUTPUT_IS_LIST = (True, False)
    FUNCTION = 'make_range'
    CATEGORY = icons.get('Comfyroll/List')

    def make_range(self, start, end, step, max_values_per_loop, operation, decimal_places, ignore_first_value, loops, ping_pong):
        show_help = 'https://github.com/Suzie1/ComfyUI_Comfyroll_CustomNodes/wiki/List-Nodes#cr-float-range-list'
        range_values = list()
        for i in range(loops):
            if end < start and step > 0:
                step = -step
            current_range = list(np.arange(start, end + step, step))
            if operation == 'sin':
                current_range = [math.sin(value) for value in current_range]
            elif operation == 'cos':
                current_range = [math.cos(value) for value in current_range]
            elif operation == 'tan':
                current_range = [math.tan(value) for value in current_range]
            current_range = [round(value, decimal_places) for value in current_range]
            if ping_pong:
                if i % 2 == 1:
                    if ignore_first_value:
                        current_range = current_range[:-1]
                    current_range = current_range[:max_values_per_loop]
                    range_values += reversed(current_range)
                else:
                    if ignore_first_value:
                        current_range = current_range[1:]
                    current_range = current_range[:max_values_per_loop]
                    range_values += current_range
            else:
                if ignore_first_value:
                    current_range = current_range[1:]
                current_range = current_range[:max_values_per_loop]
                range_values += current_range
        return (range_values, show_help)
