# Documentation
- Class name: PrimitiveInt
- Display name: Int
- Category: utils/primitive
- Output node: False
- Repo Ref: https://github.com/comfyanonymous/ComfyUI

The Int node holds a single integer value and outputs it as an INT primitive. It is used to define or pass a constant integer within a workflow. The value can be set directly via the input field or connected from another node.

# Input types
## Required
- value
    - The integer value to be stored and output by this node.
    - Comfy dtype: INT

# Output types
- INT
    - The integer value passed through from the input, available for connection to other nodes.
    - Comfy dtype: INT
