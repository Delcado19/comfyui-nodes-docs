# Documentation
- Class name: PrimitiveString
- Display name: Text String
- Category: utils/primitive
- Output node: False
- Repo Ref: https://github.com/comfyanonymous/ComfyUI

A utility node that holds and passes through a raw string value. It outputs the same string provided as input, acting as a primitive text constant or passthrough in a workflow.

# Input types
## Required
- value
    - The string value to be stored or forwarded.
    - Comfy dtype: STRING

# Output types
- STRING
    - The same string value passed through from the input.
    - Comfy dtype: STRING
