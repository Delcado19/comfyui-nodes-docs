# Documentation
- Class name: FloatToInt
- Category: math/conversion
- Output node: False
- Repo Ref: https://github.com/evanspearman/ComfyMath

The FloatToInt node converts floating-point numbers to their integer equivalents. It plays a critical role in scenarios requiring precise decimal-to-integer conversion, facilitating operations that require integer input.

# Input types
## Required
- a
    - The 'a' parameter is the floating-point number the node converts to an integer. It is essential to the node's operation, as it directly affects the output, determining the starting value of the conversion process.
    - Comfy dtype: FLOAT
    - Python dtype: float

# Output types
- result
    - The 'result' output is the integer representation of the input floating-point number. It marks the result of the conversion process, providing an integer that can be used in subsequent integer-based calculations.
    - Comfy dtype: INT
    - Python dtype: int

# Usage tips
- Infra type: CPU
