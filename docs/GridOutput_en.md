# Documentation
- Class name: GridOutput
- Category: ♾️Mixlab/Layer
- Output node: False
- Repo Ref: https://github.com/shadowcz007/comfyui-mixlab-nodes.git

This node class encapsulates the functionality for processing and outputting grid coordinates and dimensions, facilitating operations and positioning in grid-based systems.

# Input types
## Required
- grid
    - The grid parameter represents the coordinates and dimensions of elements in the grid system. It is essential for determining the position and size of elements.
    - Comfy dtype: tuple[int, int, int, int]
    - Python dtype: Tuple[int, int, int, int]

# Output types
- x
    - The x coordinate of the grid, representing the horizontal position of the element within the grid.
    - Comfy dtype: int
    - Python dtype: int
- y
    - The y coordinate of the grid, representing the vertical position of the element within the grid.
    - Comfy dtype: int
    - Python dtype: int
- width
    - The width of the element in the grid, defining its horizontal extent.
    - Comfy dtype: int
    - Python dtype: int
- height
    - The height of the element in the grid, defining its vertical extent.
    - Comfy dtype: int
    - Python dtype: int
