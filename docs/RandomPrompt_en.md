# Documentation
- Class name: RandomPrompt
- Category: ♾️Mixlab/Prompt
- Output node: True
- Repo Ref: https://github.com/shadowcz007/comfyui-mixlab-nodes.git

The RandomPrompt node aims to generate diverse prompt sets by combining mutable and immutable text elements. It operates by splitting provided prompts into individual words or phrases, then reassembling them in various combinations. The node also provides functionality to randomly sample a subset of generated prompts based on a specified maximum count, or return all prompts. The process is designed to create a wide range of text inputs for further processing or analysis.

# Input types
## Required
- max_count
    - The 'max_count' parameter determines the maximum number of prompts to generate or sample. It plays a key role in controlling output size and ensuring the node's execution aligns with the desired level of diversity while avoiding overwhelming downstream processes.
    - Comfy dtype: INT
    - Python dtype: int
- mutable_prompt
    - The 'mutable_prompt' is a string that users can change or combine with other prompts during node operation. It is important because it forms the basis for variability in generated prompts, allowing a wide range of potential outputs.
    - Comfy dtype: STRING
    - Python dtype: str
- immutable_prompt
    - The 'immutable_prompt' is a fixed string that remains unchanged during node execution. It serves as a constant element merged with the mutable prompt to produce consistent structure in the final prompt set.
    - Comfy dtype: STRING
    - Python dtype: str
## Optional
- random_sample
    - The 'random_sample' parameter indicates whether the node should randomly select a subset of generated prompts. When enabled, it introduces an element of randomness to the selection process, which can be beneficial for applications requiring diverse but limited prompt sets.
    - Comfy dtype: COMBO['enable', 'disable']
    - Python dtype: str
- seed
    - The 'seed' parameter initializes the random number generator in the random sampling process. When a specific seed value is provided, it ensures reproducibility of results, allowing the node to achieve consistent outcomes across multiple executions.
    - Comfy dtype: any_type
    - Python dtype: int

# Output types
- prompts
    - The 'prompts' output is a list of text prompts generated by combining mutable and immutable text elements. It represents the primary result of node execution and is important for applications using these prompts for further processing or analysis.
    - Comfy dtype: STRING
    - Python dtype: List[str]
