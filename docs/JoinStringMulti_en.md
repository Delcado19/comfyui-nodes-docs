# Documentation
- Class name: JoinStringMulti
- Display name: Join String Multi
- Category: KJNodes/text
- Output node: False
- Repo Ref: https://github.com/kijai/ComfyUI-KJNodes

Creates single string, or a list of strings, from  
multiple input strings.  
You can set how many inputs the node has,  
with the **inputcount** and clicking update.

# Input types
## Required
- inputcount
    - Comfy dtype: INT
    - Default: `2`
- string_1
    - Comfy dtype: STRING
    - Default: `""`
- delimiter
    - Comfy dtype: STRING
    - Default: `" "`
- return_list
    - Comfy dtype: BOOLEAN
    - Default: `false`
## Optional
- string_2
    - Comfy dtype: STRING
    - Default: `""`

# Output types
- string
    - Comfy dtype: STRING
