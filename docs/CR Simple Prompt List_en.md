# Documentation

- Category: Comfyroll/Animation/Legacy

CR_SimplePromptList is a node designed to efficiently aggregate and manage prompt lists. Its purpose is to stack multiple prompts into a single list, which can be used for various downstream processing tasks. The node’s function focuses on merging text prompts, emphasizing its role in simplifying prompt management for animation workflows.
