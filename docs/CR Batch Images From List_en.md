# Documentation
- Class name: CR_MakeBatchFromImageList
- Category: Comfyroll/List/Utils
- Output node: False
- Repo Ref: https://github.com/RockOfFire/ComfyUI_Comfyroll_CustomNodes

The CR_MakeBatchFromImageList node aims to receive a series of images and efficiently merge them into a batch tensor. This process simplifies the processing of image data, making the image processing workflow more efficient. The node's function plays a key role in optimizing the conversion from single image files to a format suitable for batch operations, thereby improving the overall performance and efficiency of the image processing pipeline.
