- Output node: False

This node class encapsulates thresholding operations within the HSV color space, capable of creating binary masks based on specific hue, saturation, or brightness ranges. This is critical for segmenting and isolating regions of interest in images, facilitating various image processing tasks such as object recognition and noise reduction.

# Input types
## Required
- image
    - The image parameter is essential for the HSV thresholding process, as it provides the source data for threshold segmentation. It influences the entire operation by determining which regions of the image will be processed by the mask criteria.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor
- low_threshold
    - The low threshold parameter sets the lower bound for HSV channel values, playing a crucial role in defining the range of pixel values to be considered for masking.
    - Comfy dtype: FLOAT
    - Python dtype: float
- high_threshold
    - The high threshold parameter establishes the upper bound for HSV channel values, and together with the low threshold, determines the final mask.
    - Comfy dtype: FLOAT
    - Python dtype: float
- hsv_channel
    - The hsv_channel parameter indicates which channel in the HSV color space will be used for thresholding, significantly impacting the resulting mask and its applicability to the image processing task at hand.
    - Comfy dtype: COMBO['hue', 'saturation', 'value']
    - Python dtype: str

# Output types
- result
    - The result parameter represents the binary mask generated by applying the HSV threshold criteria, serving as an important tool for further image analysis and manipulation.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor

class HSVThresholdMask:

    def __init__(self):
        pass

    @classmethod
    def INPUT_TYPES(s):
        return {'required': {'image': ('IMAGE',), 'low_threshold': ('FLOAT', {'default': 0.2, 'min': 0, 'max': 1, 'step': 0.1}), 'high_threshold': ('FLOAT', {'default': 0.7, 'min': 0, 'max': 1, 'step': 0.1}), 'hsv_channel': (['hue', 'saturation', 'value'],)}}
    RETURN_TYPES = ('IMAGE',)
    FUNCTION = 'hsv_threshold'
    CATEGORY = 'postprocessing/Masks'

    def hsv_threshold(self, image: torch.Tensor, low_threshold: float, high_threshold: float, hsv_channel: str):
        (batch_size, height, width, _) = image.shape
        result = torch.zeros(batch_size, height, width)
        if hsv_channel == 'hue':
            channel = 0
            (low_threshold, high_threshold) = (int(low_threshold * 180), int(high_threshold * 180))
        elif hsv_channel == 'saturation':
            channel = 1
            (low_threshold, high_threshold) = (int(low_threshold * 255), int(high_threshold * 255))
        elif hsv_channel == 'value':
            channel = 2
            (low_threshold, high_threshold) = (int(low_threshold * 255), int(high_threshold * 255))
        for b in range(batch_size):
            tensor_image = (image[b].numpy().copy() * 255).astype(np.uint8)
            hsv_image = cv2.cvtColor(tensor_image, cv2.COLOR_RGB2HSV)
            mask = cv2.inRange(hsv_image[:, :, channel], low_threshold, high_threshold)
            tensor = torch.from_numpy(mask).float() / 255.0
            result[b] = tensor
        return (result,)
