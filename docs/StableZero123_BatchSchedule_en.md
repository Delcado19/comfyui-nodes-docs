# Documentation
- Class name: StableZero123_BatchSchedule
- Category: KJNodes/experimental
- Output node: False
- Repo Ref: https://github.com/kijai/ComfyUI-KJNodes.git

The `encode` method of the StableZero123_BatchSchedule class is responsible for processing images and generating latent representations. It accepts various parameters such as vision model output, initial image, VAE model, and additional settings like width, height, and batch size. This method also performs angular interpolation based on provided azimuth and elevation points, applying different easing functions to achieve smooth transitions. The final output includes positive and negative conditioning data and latent image representations.
