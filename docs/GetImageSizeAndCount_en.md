# Documentation
- Class name: GetImageSizeAndCount
- Display name: Get Image Size & Count
- Category: KJNodes/image
- Output node: False
- Repo Ref: https://github.com/kijai/ComfyUI-KJNodes

and passes it through unchanged.

- count
