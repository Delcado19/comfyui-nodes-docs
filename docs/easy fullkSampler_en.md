# Documentation
- Class name: samplerFull
- Category: EasyUse/Sampler
- Output node: True
- Repo Ref: https://github.com/yolain/ComfyUI-Easy-Use.git

The `samplerFull` node aims to provide a comprehensive sampling framework for generating high-quality images. It leverages the power of advanced sampling techniques to create visually coherent and diverse outputs. The primary function of this node is to facilitate the generation of images that are consistent with input conditions and constraints, ensuring rich result diversity. By abstracting the sampling process, this node emphasizes the creation of images that are both aesthetically pleasing and technically accurate, making a significant contribution to the system's creative and research-oriented goals.

# Input types
## Required
- pipe
    - The `pipe` parameter is the primary input for the `samplerFull` node, providing the necessary data and settings for the sampling process. It includes model information, conditioning data, and configuration parameters that guide the generation of the final image. This parameter is crucial as it directly influences the quality and characteristics of the output image.
    - Comfy dtype: PIPE_LINE
    - Python dtype: Dict
- steps
    - The `s steps` parameter defines the number of iterations the sampling process will undergo. It is a critical factor in determining the level of refinement in the generated image. Generally, more steps result in a more detailed and refined image, as the model has more opportunities to adjust and improve image quality.
    - Comfy dtype: INT
    - Python dtype: int
- cfg
    - The `cfg` parameter adjusts the configuration settings of the sampling process, influencing various aspects such as noise levels and model behavior. It plays a significant role in tuning the output to meet specific requirements or desired results, allowing for a customized approach to image generation.
    - Comfy dtype: FLOAT
    - Python dtype: float

# Output types
- pipe
    - The `pipe` output of the `samplerFull` node contains the comprehensive data and settings updated and refined during the sampling process. It includes the final image, model information, and any other relevant details necessary for further processing or analysis.
    - Comfy dtype: PIPE_LINE
    - Python dtype: Dict
- image
    - The `image` output represents the final image generated by the sampling process. These images are a direct manifestation of the input parameters and settings, demonstrating the `samplerFull` node's capability to create visually appealing and contextually relevant outputs.
    - Comfy dtype: IMAGE
    - Python dtype: List[torch.Tensor]
