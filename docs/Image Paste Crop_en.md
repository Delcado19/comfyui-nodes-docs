# Documentation
- Class name: WAS_Image_Paste_Crop
- Category: WAS Suite/Image/Process
- Output node: False
- Repo Ref: https://github.com/WASasquatch/was-node-suite-comfyui

The WAS_Image_Paste_Crop node is designed to integrate and manipulate images by pasting a secondary image onto a primary image, then cropping and blending them. It allows fine-tuning of blending and sharpening effects to achieve seamless image integration.

# Input types
## Required
- image
    - The primary image onto which the secondary image will be pasted. It serves as the foundation for the entire operation, and its properties influence the final output.
    - Comfy dtype: IMAGE
    - Python dtype: PIL.Image.Image
- crop_image
    - The secondary image that will be pasted onto the primary image. It is resized and positioned according to the provided crop data.
    - Comfy dtype: IMAGE
    - Python dtype: PIL.Image.Image
- crop_data
    - Defines the size and position of the crop operation. It is crucial for determining how the secondary image is integrated into the primary image.
    - Comfy dtype: CROP_DATA
    - Python dtype: Tuple[int, Tuple[int, int, int, int]]
## Optional
- crop_blending
    - Controls the amount of blending between the images. Higher values produce smoother transitions between the images, while lower values make the secondary image more prominent.
    - Comfy dtype: FLOAT
    - Python dtype: float
- crop_sharpening
    - Adjusts the level of sharpening applied to the secondary image. Increasing this value enhances image details, which is useful for making the pasted image sharper.
    - Comfy dtype: INT
    - Python dtype: int

# Output types
- result_image
    - The final processed image after the paste and crop operation. It represents the seamless integration of the secondary image with the primary image along with the applied effects.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor
- result_mask
    - The mask generated by the crop operation, which can be used for further image processing tasks or to isolate the pasted image from the primary image.
    - Comfy dtype: IMAGE
    - Python dtype: torch.Tensor
